AGITA Ginástica - Plataforma Institucional
Este projeto consiste na refatoração completa do portal da Academia AGITA, focada em Ginástica Rítmica. O objetivo é substituir o rascunho anterior por uma aplicação moderna, eliminando redundâncias visuais e adotando metodologias ágeis de desenvolvimento.

🚀 Tecnologias Utilizadas
Frontend: React.

Backend: NestJS (Arquitetura escalável).

ORM: Prisma (Gestão de banco de dados e migrations).

Banco de Dados: PostgreSQL (Rodando via Docker).

Infraestrutura: Docker para padronização do ambiente de desenvolvimento.

📋 Funcionalidades Principais
Home Otimizada: Apresentação da missão e visão, com integração direta para redes sociais (Instagram/WhatsApp) e localização.

Galeria de Atletas: Espaço dedicado para fotos e conquistas das ginastas.

Gestão de Eventos: Calendário de competições e festivais da academia.

Página Sobre: História da AGITA e perfil dos responsáveis técnicos.
